# CardReader-IO
ほっくんハウスのカードリーダーを使用するためのライブラリです。(識別子はFELICA IDMを使用する簡易的なものです)

## 送信コマンド一覧

1. PING (カードリーダーからPONGが返ってきます。)
2. INIT (カードリーダーを使用可能な状態にする。ロックからの復帰時も使用する必要があります。)
3. ERROR (カードリーダーにエラー送信します。赤色に光ります。)
4. LOCK (カードリーダーをロックします。青く光ります。)
5. BEEP (カードリーダーからビープ音を鳴らします。)
6. BEEP2 (カードリーダーからビープ音を2回鳴らします。)
7. GETIDM (現在のIDMを送信します。)

## 受信コマンド一覧

1. ERR:UNKNOWN_CMD (存在しないコマンドが指定されました。)
