# CardReader-IO
ほっくんハウスのカードリーダーを使用するためのライブラリです。(識別子はFELICA IDMを使用する簡易的なものです)

# 送信コマンド一覧
・PING (カードリーダーからPONGが返ってきます。)
・INIT (カードリーダーを使用可能な状態にする。ロックからの復帰時も使用する必要があります。)
・ERROR (カードリーダーにエラー送信します。赤色に光ります。)
・LOCK (カードリーダーをロックします。青く光ります。)
・BEEP (カードリーダーからビープ音を鳴らします。)
・BEEP2 (カードリーダーからビープ音を2回鳴らします。)
・GETIDM (現在のIDMを送信します。)

# 受信コマンド一覧
・ERR:UNKNOWN_CMD (存在しないコマンドが指定されました。)
